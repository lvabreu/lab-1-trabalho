# Implementações usando árvores binárias

### Conceitos e tipos de árvores

Uma árvore de busca binária é uma estrutura hierárquica onde cada nó tem no máximo dois filhos. O nó raiz é o ponto inicial, com filhos organizados à esquerda e direita. Suporta operações como busca, inserção, remoção, e localização de mínimos, máximos, antecessores e sucessores.
